Class ListValueContainerImpl

public ListValueContainerImpl

Container class for values held explicitly. Right now, we are allowing any old Java object as values. It may be that we will want to be more restrictive (e.g. only String, int/Integer, float/Float, etc) objects to be allowed to be held by this class. In this fashion, we can automatically not have to worry about the user passing us wierd objects to be held by the quantity..and allows us in the future we to be checking that the passed information conforms to the declared dataType of the parent quantity.
