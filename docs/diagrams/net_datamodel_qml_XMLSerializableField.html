Class XMLSerializableField

public XMLSerializableField

XMLSerializableField works in tandem with XMLSerializableObject to provide flexible XML serialization in the QML package. The XMLSerializableField class works as a minimal wrapper around those objects which are fields of an XMLSerializableObject. Various types of XMLSerializableFields are supported, and these include fields which are either serialized as PCDATA, CDATA, child XML nodes or attributes. If the object is to be serialized as a child node, it needs to inherit from the XMLSerializableObject class.
