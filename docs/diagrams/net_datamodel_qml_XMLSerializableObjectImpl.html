Class XMLSerializableObjectImpl

public XMLSerializableObjectImpl

Underlying class for most QML package objects, this class enables flexible XML serialization for all classes which need it. This class manages the storage, retrieval and writing out the XML-serializable fields of various QML classes. The 2 parts that make this work in XMLSerializableObjectImpl are 1) a lookup table of key/value pairs in fieldHash and 2) a list containing the proper order of the fields. A separate XMLSerializableField class manages the aspects of these (child) fields.
