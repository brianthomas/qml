Class QuantityContainerImpl

public QuantityContainerImpl

This is a simple container class which holds quantities. It is used internally by other quantities to hold a list of one or more quantities which will be serialized as a group of child objects. It is not intended for external (outside of the Quantity package) use as it can't be cast as a Quantity (this is intentional..this class models containers which aren't seperable from the parent quantity). For most people, you should use a CompositeQuantity instead.
